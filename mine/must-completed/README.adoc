= ERP

== 登录 新增或者加入企业。

   新增 --> 成为企业的普通人
   加入企业 --> 企业任何一个人同意加入就可以
   成为企业的管理人员 --> 企业管理的人 可以做企业统计，企业账户  企业金额等。 
   只能退出企业 加入新的企业
   当企业没人了 可以删除这个企业


== 企业人员管理

这是角色

. 管理员
.. 所有功能
. 订单管理人
.. 所有订单查看确认
. 采购人(去其他企业采购,下采购单)
.. 采购功能(是否要审批)
. 确认发货的人
.. 确认订单发货
. 销售人员(让别人来采购,自己也可以录入单子)
.. 查看自己的单

== 下单

就是下单
从合作商的商品中选取商品下单
-> 生成订单
-> 企业的订单管理人员操作确认订单
-> 企业的发货人确认发货
-> 订单完成
